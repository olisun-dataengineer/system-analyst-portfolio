# UI-001. Авторизация и регистрация

## **Назначение:**
Экранная форма используется для входа пользователей в систему и регистрации нового ученика.

Форма содержит два состояния:

* вход в систему;
* регистрация ученика.

## Доступные роли:

* Вход в систему: ученик, репетитор, администратор.
* Регистрация: ученик.

## **Связанные функциональные требования:**  
FR-001, FR-002, FR-008, FR-027, FR-038, FR-039, FR-040, FR-041

## Состояния экранной формы

| Состояние           | Описание                                                |
| ------------------- | ------------------------------------------------------- |
| Вход в систему      | Пользователь вводит email и пароль для входа в систему. |
| Регистрация ученика | Новый ученик вводит данные для создания учётной записи. |

---

## Переход на экранную форму

Пользователь попадает на экранную форму:

* при открытии системы без активной авторизации;
* после выхода из учётной записи;
* при переходе по ссылке на страницу входа.

## Правила работы и ограничения формы

- По умолчанию отображается состояние «Вход».
- В состоянии «Вход» отображаются только элементы входа.
- В состоянии «Регистрация ученика» отображаются только элементы регистрации ученика.
- Самостоятельная регистрация доступна только ученику.
- Репетитор и администратор не могут самостоятельно зарегистрироваться через эту форму.
- Если пользователь уже авторизован, форма авторизации и регистрации не отображается.
- После успешного входа пользователь перенаправляется в раздел, доступный его роли.
- После успешной регистрации система создаёт учётную запись ученика.
- При выборе «Запомнить меня» система сохраняет авторизацию пользователя на устройстве.
- При переходе по ссылке «Забыли пароль?» пользователь переходит к восстановлению пароля.
- Пароль не должен отображаться в открытом виде без действия пользователя.
- Заблокированный пользователь не может войти в систему.
- Регистрация по заблокированному email или номеру телефона запрещена.

## Валидация данных и проверки системы

| Поле / действие                    | Правило проверки                                                                | Сообщение об ошибке                                                              |
| ---------------------------------- | ------------------------------------------------------------------------------- | -------------------------------------------------------------------------------- |
| Email                              | Поле обязательно для заполнения. Email должен быть указан в корректном формате. | Введите корректный email.                                                        |
| Пароль при входе                   | Поле обязательно для заполнения.                                                | Введите пароль.                                                                  |
| Имя                                | Поле обязательно для заполнения.                                                | Введите имя.                                                                     |
| Фамилия                            | Поле обязательно для заполнения.                                                | Введите фамилию.                                                                 |
| Пароль при регистрации             | Пароль должен содержать не менее 8 символов.                                    | Пароль должен содержать не менее 8 символов.                                     |
| Подтверждение пароля               | Значение должно совпадать с полем «Пароль».                                     | Пароли не совпадают.                                                             |
| Телефон                            | Поле должно содержать номер телефона в корректном формате.                      | Введите корректный номер телефона.                                               |
| Согласие с условиями               | Чекбокс должен быть отмечен перед регистрацией.                                 | Необходимо согласиться с условиями использования и политикой конфиденциальности. |
| Вход в систему                     | Пользователь с указанными данными должен существовать в системе.                | Неверный email или пароль.                                                       |
| Вход заблокированного пользователя | Пользователь не должен иметь статус `blocked`.                                  | Учётная запись заблокирована.                                                    |
| Регистрация ученика                | Email или телефон не должны принадлежать заблокированному пользователю.         | Регистрация с указанным email или номером телефона недоступна.                   |
| Регистрация ученика                | Email или телефон не должны быть уже зарегистрированы в системе.                | Пользователь с таким email или номером телефона уже существует.                  |

## Результат успешного действия

| Действие                                  | Результат                                                             |
| ----------------------------------------- | --------------------------------------------------------------------- |
| Успешный вход                             | Пользователь авторизуется и переходит в раздел, доступный его роли.   |
| Успешная регистрация ученика              | Система создаёт учётную запись ученика.                               |
| Успешный запрос восстановления пароля     | При переходе по ссылке «Забыли пароль?» пользователь переходит к форме восстановления пароля. |
| Выбор «Запомнить меня» при успешном входе | Система сохраняет авторизацию пользователя на устройстве.             |
