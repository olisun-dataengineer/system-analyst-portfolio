# Online Tutor Booking System

## Описание проекта

Система онлайн-записи к репетитору позволяет ученикам самостоятельно выбирать свободное время для занятия, записываться на уроки и отменять записи.

Репетитор получает возможность управлять расписанием, создавать свободные временные слоты и просматривать список запланированных занятий.

## Проблема

Сейчас запись на занятия происходит вручную через мессенджеры. Репетитор тратит время на переписку, проверку расписания и ведение записей в таблицах.

Из-за ручного процесса возможны ошибки:
- двойная запись на одно время;
- потеря информации о занятии;
- забытые отмены;
- длительное согласование времени.

## Цель проекта

Автоматизировать процесс записи учеников на занятия и снизить количество ручных действий репетитора.

## Основные пользователи

- Ученик
- Репетитор
- Администратор системы

## Основные возможности

### Ученик

- регистрация и вход в личный кабинет;
- просмотр доступных временных слотов;
- запись на занятие;
- отмена занятия;
- просмотр истории занятий.

### Репетитор

- создание свободных временных слотов;
- просмотр расписания;
- отмена занятия;
- просмотр списка учеников.

### Администратор

- управление пользователями;
- просмотр всех занятий;
- блокировка пользователей при необходимости.
